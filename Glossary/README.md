# DANH MỤC THUẬT NGỮ (GLOSSARY) - DỰ ÁN E-OFFICE HÒA PHÁT

| Thuật ngữ | Định nghĩa |
| :--- | :--- |
| **Workflow** | Quy trình xử lý công việc theo nhiều bước và cấp phê duyệt (Fe-04, Fe-05). |
| **Công văn** | Tài liệu hành chính được tạo và xử lý trong hệ thống (UC-01, UC-05). |
| **Phê duyệt** | Quá trình chấp thuận hoặc từ chối một yêu cầu (UC-02, UC-03). |
| **Hồ sơ số** | Tài liệu được lưu trữ dưới dạng điện tử tập trung (Fe-08). |
| **Metadata** | Thông tin mô tả dữ liệu như ngày tạo, người tạo, loại tài liệu (UC-06, Fe-09). |
| **AI** | Công nghệ hỗ trợ tự động phân loại và gợi ý xử lý dữ liệu (UC-09, UC-10). |
| **Stakeholder** | Các bên liên quan đến hệ thống như lãnh đạo, nhân viên, IT (Section 2.4). |
| **Use Case** | Mô tả tương tác giữa người dùng và hệ thống (UC-01 đến UC-10). |
| **Feature** | Đặc trưng chức năng ở mức cao của hệ thống (Fe-01 đến Fe-25). |
| **Notification** | Thông báo hệ thống gửi đến người dùng tức thời (Fe-14, UC-08). |
| **Real-time** | Cập nhật và phản hồi thông tin ngay lập tức (BR-09). |
